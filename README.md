# 五線譜 + 技術指標 · 美股 / 台股查詢

樂活五線譜風格的股票走勢查詢網站，疊加常見技術指標，並以多指標綜合分數明確輸出 **BUY / HOLD / SELL** 建議。

## 功能

- **五線譜**：3.5 / 5 / 10 年線性回歸 ± 1σ / ± 2σ（樂觀價、相對高價、趨勢價、相對低價、悲觀價）
- **技術指標**：MA5 / MA20 / MA60、RSI(14)、MACD(12, 26, 9)、KD(9, 3, 3)
- **分析師目標價**：最低 / 平均 / 中位 / 最高，含相對現價漲跌幅、推薦標籤、分析師人數
- **相關新聞**：Yahoo Finance 最新 10 則（含縮圖、發行單位、相對時間）
- **綜合訊號**：6 項指標加總（範圍 −9 ~ +9）→ STRONG BUY / BUY / HOLD / SELL / STRONG SELL，含每項指標明細
- 同時支援美股（`AAPL`、`TSLA`...）與台股（`2330`、`0050`...，自動補 `.TW`；找不到再試 `.TWO` 上櫃）

## 安裝

```bash
pip install -r requirements.txt
```

## 啟動

```bash
python app.py
# 開瀏覽器：http://127.0.0.1:5001
```

## 評分規則

| 指標 | +2 | +1 | 0 | −1 | −2 |
|------|----|----|---|----|----|
| 五線譜 z-score | z ≤ −1.5 | z ≤ −0.5 | −0.5 < z < 0.5 | z ≥ 0.5 | z ≥ 1.5 |
| RSI(14) | — | < 30 | 30–70 | > 70 | — |
| MACD 柱狀體 | — | 正且擴張 | 其他 | 負且擴張 | — |
| 均線排列 | — | 多頭排列 | 盤整 | 空頭排列 | — |
| KD | 低檔黃金交叉 | 黃金交叉或超賣 | 中性 | 死亡交叉或超買 | 高檔死亡交叉 |
| 分析師共識 | strong_buy | buy | hold | sell | strong_sell |

**綜合分數區間 → 建議**

| 分數 | 建議 |
|------|------|
| ≥ +5 | STRONG BUY |
| +2 ~ +4 | BUY |
| −1 ~ +1 | HOLD |
| −2 ~ −4 | SELL |
| ≤ −5 | STRONG SELL |
